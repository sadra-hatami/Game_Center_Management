# 📖 About

**Game Center Management** is a browser-based management system developed to simplify the daily operation of gaming centers.

The application allows operators to manage multiple gaming devices, calculate play sessions, determine end times automatically, select the number of controllers, and instantly calculate service costs through a clean and easy-to-use interface.

Designed with simplicity and efficiency in mind, the project helps streamline routine game center management tasks.

---

# ✨ Features

* 🎮 Manage multiple gaming devices
* ⏰ Automatic end-time calculation
* 🎮 Controller quantity selection
* 💰 Automatic cost calculation
* 📊 Organized table-based interface
* ⚡ Fast and lightweight
* 🎨 Clean and intuitive user interface
* 📱 Responsive design
* 💻 Cross-platform compatibility
* 🌐 Runs directly in any modern web browser

---

# 🎯 Project Goals

The goal of this project is to provide a simple and efficient management solution for gaming centers by automating time tracking and pricing calculations while reducing manual work.
